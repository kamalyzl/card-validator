Validador de Tarjeta de Crédito

Descripción

ValidCard es una librería creada para facilitarte la utilidad al momento de querer validar una tarjeta de crédito.

Considerar

  1. Solo puedes ingresar números de 16 dígitos
  2. Si tienes una base de datos, puedes anexarla para realizar tus validaciones.
