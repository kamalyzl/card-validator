Estructura Github

Contenido
- public/js/index.js - contiene la libreria en su versión ES5.
- src/index.js - contiene la libreria en su versión ES6. public/.
  
              public/
              ├── js/
              │   ├── index.js
              │   ├
              src/
              ├── index.js
Instalación
Pasos
- Descargar el archivo index.js.
- Copiar el archivo index.js a la carpeta donde tienes los archivos js.
- Agregar archivo en un documento html como un script.
            
              <script src="index.js"></script>

Uso

Métodos
- Ingresar el número de tarjeta, fecha, nombre y CVV como parámetro.
- El resultado será un booleano (true o false).
  
          card.validCard('');
        
          card.validName('');
    
          card.validCvv('');
   
          card.validExp('');
